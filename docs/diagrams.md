## 3. 检索策略流程图

```
┌──────────────────┐
│ "Day1要交什么？" │
└──────────────────┘
          │
          ▼
┌──────────────────────────────────────────────────────────────────────────────────┐
│ tokenize(query)                                                                  │
│   输入: "Day1要交什么？"                                                         │
│   输出: ["day1","要","交","什么","要交","交什","什么"]                            │
└──────────────────────────────────────────────────────────────────────────────────┘
          │
          ▼
┌──────────────────────────────────────────────────────────────────────────────────┐
│ 对每个chunk计算score                                                             │
│                                                                                  │
│   chunk = {"id":"faq-02", "title":"6类证据文件", "content":"..."}                 │
│                                                                                  │
│   chunk_tokens = tokenize(title + content)                                       │
│   hits = query_tokens ∩ chunk_tokens                                             │
│   score = hits / len(query_tokens)                                               │
│   if title命中: score += 0.3                                                     │
└──────────────────────────────────────────────────────────────────────────────────┘
          │
          ▼
┌──────────────────────────────────────────────────────────────────────────────────┐
│ 过滤(score>=0.25) → 排序(降序) → 取top_k=3                                       │
└──────────────────────────────────────────────────────────────────────────────────┘
          │
          ▼
┌──────────────────────────────────────────────────────────────────────────────────┐
│ 结果: [{"id":"faq-02","score":0.85}, {"id":"faq-06","score":0.45}]               │
└──────────────────────────────────────────────────────────────────────────────────┘
```
