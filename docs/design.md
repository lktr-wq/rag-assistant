# RAG 课程助手 — 设计文档

## 数据流

```
course-faq.md → parse_faq() → retrieve() → Prompt 拼接 → LLM → 输出
     ↓              ↓              ↓              ↓
  10条FAQ      切片为chunk      关键词匹配      检索结果+问题    生成回答+来源
```

## 检索策略

### 分词方案

中文按 2-gram + 英文按空格小写：
- 中文字符提取正则：`[\u4e00-\u9fff]`
- 英文单词提取正则：`[a-z0-9]+`
- 中文 2-gram：相邻两字组合，捕捉词级语义

### 评分公式

```
score = (query token 命中数 / query token 总数) + 标题加权
```

- 标题加权：query token 出现在标题中，额外 +0.3
- 阈值过滤：score < 0.15 的结果丢弃

### 返回结果

按 score 降序排列，返回 top_k=3 个 chunk

## 拒答逻辑

三道防线：

1. **空输入**：query 为空或纯空格 → 提示"请输入您的问题"
2. **无结果**：retrieve() 返回空列表 → 拒答
3. **低分**：最高分 < 0.15 → 拒答

拒答回答："资料中没有找到依据。本助手仅能回答 Day1 AI Native 训练营相关问题。"

## Prompt 模板

### System 指令

```
你是 Day1 AI Native 训练营的课程助手。

规则（必须严格遵守）：
1. 只基于下方【课程资料】回答问题，不要使用任何外部知识
2. 如果资料中没有相关内容，直接回答"资料中没有找到依据"
3. 回答末尾必须注明引用的来源编号，格式：来源: [faq-XX]
4. 如果涉及多个 FAQ 条目，列出所有相关来源
5. 用中文回答，语言简洁准确
```

### User 消息

```
【课程资料】
---
【faq-XX】标题
内容
---

【用户问题】
问题内容

请基于课程资料回答，并在末尾注明来源。
```
