# RAG 课程助手 — 规格说明

## 目标（Goals）

1. 用户输入 Day1 训练营相关问题，CLI 输出基于 `course-faq.md` 的回答和来源编号 `[faq-XX]`
2. 资料外问题必须拒答，回答"资料中没有找到依据"
3. 回答必须注明来源，格式：来源: [faq-XX]
4. 支持跨段落问题，返回多个相关来源
5. 空输入时提示用户"请输入您的问题"

## 非目标（Non-Goals）

1. 不做 Web UI（CLI 工具）
2. 不做用户登录（单用户本地运行）
3. 不引入数据库或外部存储（文件系统即数据源）
4. 不支持 PDF 上传（仅支持 Markdown 格式）
5. 不支持除 `course-faq.md` 之外的资料源

## 验收标准（Acceptance Criteria）

1. `python tests/test_basic.py` 全部测试通过
2. 输入"Day1要交什么？"，输出包含 6 类证据文件描述，来源为 `[faq-02]`
3. 输入"什么是可复核交付？"，输出包含定义和核心原则，来源为 `[faq-01]`
4. 输入"奖学金政策是什么？"，输出"资料中没有找到依据"
5. 输入空字符串，输出"请输入您的问题"
