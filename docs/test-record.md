## 5. questions.json 样例覆盖说明

`tests/questions.json` 是课程提供的问题样例集，不是自动化测试脚本。本项目将它作为功能测试设计依据，用来覆盖正确匹配、跨段落、资料外、混淆、空输入五类问题。

| 类型 | questions.json 示例 | test-record 覆盖情况 |
|------|--------------------|---------------------|
| 正确匹配 | `Day1要交什么？` | 已覆盖：2.1 正确匹配问题 |
| 正确匹配 | `什么是可复核交付？` | 已通过检索与回答链路验证，来源应为 `[faq-01]` |
| 正确匹配 | `ai-log的五字段是什么？` | 已作为补充手测样例，来源应为 `[faq-03]` |
| 跨段落 | `Spec中的非目标和AI过度设计有什么关系？` | 已覆盖：2.2 跨段落问题 |
| 跨段落 | `证据包和工具链有什么关系？` | 已作为补充手测样例，期望返回 `[faq-02]`、`[faq-06]` 等相关来源 |
| 资料外 | `奖学金政策是什么？` | 已覆盖：2.3 资料外问题 |
